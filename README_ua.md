### Огляд

Digital Signature App — це застосунок Windows Forms, розроблений на C# (.NET Framework 4.7.2), який демонструє процес створення цифрового підпису документів за допомогою криптографічного алгоритму ECDSA (Elliptic Curve Digital Signature Algorithm).

Програма дозволяє:

* Обирати документи Microsoft Word (`.doc`, `.docx`)
* Вводити ім'я та прізвище користувача
* Генерувати цифровий підпис ECDSA
* Створювати PDF-документ з інформацією про підпис
* Експортувати дані підпису у файл `.p7s`

> Проєкт створено з навчальною та дослідницькою метою для демонстрації принципів електронного підпису та забезпечення цілісності документів.

### 💡 Принцип роботи

1. Користувач обирає документ Word.
2. Вводить ім'я та прізвище.
3. Програма генерує пару ключів ECDSA.
4. Документ хешується за допомогою SHA-256.
5. Створюється цифровий підпис документа.
6. Формується PDF-файл, який містить:
   * Дані підписувача
   * Згенерований підпис
7. Додатково створюється файл `.p7s` з інформацією про підпис.

### ⚠️ Відмова від відповідальності

Цей проєкт створений виключно з навчальною метою.

Реалізація демонструє базові принципи цифрового підпису, але не є повноцінним рішенням для використання в державних або комерційних системах електронного документообігу. Для практичного застосування необхідні сертифікати, захищене зберігання ключів та механізми перевірки підписів.
